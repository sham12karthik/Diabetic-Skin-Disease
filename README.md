🧠 Diabetic Skin Disease Detection
📌 Overview

This project is a deep learning–based system designed to detect diabetic skin diseases from images.
It classifies input images into categories such as Normal and Abnormal (Diabetic-related conditions).

🚀 Features
🖼️ Image-based disease detection
🤖 Deep learning model integration
⚡ Fast and lightweight setup
📊 Uses class mapping via CSV
🗂️ Project Structure
Diabetic-Skin-Disease/
│
├── app.py            # Main application
├── model.py          # Model loading and prediction logic
├── class_dict.csv    # Class labels mapping
├── .gitignore        # Ignored files
🛠️ Installation
1. Clone the repository
git clone https://github.com/sham12karthik/Diabetic-Skin-Disease.git
cd Diabetic-Skin-Disease
2. Install dependencies
pip install -r requirements.txt
▶️ How to Run
python app.py
📊 Model Details
Trained on skin disease dataset
Uses image classification techniques
